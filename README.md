# IPTV Player — تطبيق أندرويد (Xtream Codes)

مشغّل IPTV أصلي لنظام أندرويد، يدعم **سامسونج (لمس)** و **NVIDIA Shield Pro (ريموت/D-pad)**.

## ✨ المميزات
- اتصال **Xtream Codes** مباشر بدون قيود CORS (الطلبات تُنفّذ من كود أندرويد الأصلي).
- **الميزة الأساسية:** تشغيل الفيديو من قناة وأخذ الصوت من قناة أخرى (مشغّلان مستقلان عبر ExoPlayer).
- تبديل صيغة البث `.ts` / `.m3u8` أثناء التشغيل.
- بحث + تصنيف بالمجموعات + شعارات القنوات.
- تحكم كامل بالريموت (D-pad) واللمس.

## 📦 كيف تحصل على ملف APK (بدون تثبيت أي برنامج على جهازك)

التطبيق مهيّأ ليُبنى تلقائياً عبر **GitHub Actions**:

1. أنشئ حساب على [github.com](https://github.com) (مجاني).
2. أنشئ مستودعاً جديداً (New repository) — سمّه مثلاً `iptv-player`.
3. ارفع كل ملفات هذا المجلد إلى المستودع:
   - إما عبر زر **Add file → Upload files** وسحب كل المحتويات.
   - أو عبر Git إذا كنت تعرفه.
4. اذهب لتبويب **Actions** في المستودع → ستجد workflow باسم **Build APK**.
5. إذا لم يبدأ تلقائياً، اضغط **Run workflow**.
6. انتظر ٥–١٠ دقائق حتى تكتمل العلامة الخضراء ✅.
7. افتح نتيجة البناء → قسم **Artifacts** → نزّل **IPTVPlayer-debug**.
8. فك الضغط → ستجد ملف `app-debug.apk`.

## 📲 التثبيت

### على سامسونج
- انقل ملف `app-debug.apk` للجهاز وافتحه، واسمح بالتثبيت من مصادر غير معروفة.

### على NVIDIA Shield Pro
- استخدم تطبيق **Downloader** أو **Send Files to TV** لنقل الـ APK، أو ثبّته عبر ADB:
  ```
  adb connect <عنوان-IP-للشيلد>
  adb install app-debug.apk
  ```
- سيظهر التطبيق في صف تطبيقات Android TV.

## 🎮 التحكم بالريموت (Shield)
- **اتجاهات D-pad:** للتنقل بين القنوات والأزرار.
- **زر OK/الاختيار:** لتشغيل القناة المحددة.
- **زر Back:** يوقف التشغيل ويعود لقائمة القنوات (ضغطة ثانية للخروج).

## 🔑 طريقة الدخول
أدخل بيانات اشتراكك:
- **رابط الخادم:** مثل `http://cc.wd.business-cdn-8k.com` (مع المنفذ إن وُجد).
- **المستخدم** و **كلمة المرور**.
- اختر الصيغة الافتراضية (`.ts` أنسب لأغلب الخوادم).

## ⚙️ البناء محلياً (اختياري لمن لديه Android Studio)
```
./gradlew assembleDebug
```
الناتج: `app/build/outputs/apk/debug/app-debug.apk`
